# 🧮 Calculadora de IMC - Vanilla JS

🔗 **[Acesse o projeto aqui](https://MPicolli.github.io/calculadora-imc-vanilla-js/)**

Este é um projeto de uma calculadora de Índice de Massa Corporal (IMC) desenvolvido com tecnologias fundamentais da web. O objetivo foi aplicar conceitos de manipulação do DOM e lógica de programação com JavaScript puro.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estrutura do formulário e dos elementos da página.
* **CSS3**: Estilização moderna e responsiva.
* **JavaScript (Vanilla)**: Lógica para captura de dados, cálculo matemático e exibição dos resultados.

## 📋 Sobre o Cálculo

O IMC é calculado dividindo o peso (em kg) pela altura (em metros) ao quadrado. Como tutor de matemática, fiz questão de implementar a fórmula padrão:

$$IMC = \frac{peso}{altura^2}$$

O sistema classifica o resultado em categorias como: Baixo peso, Peso normal, Sobrepeso e Obesidade.

## 🧠 O que eu pratiquei neste projeto?

* Captura de eventos de formulário (`submit`).
* Uso de funções para organizar a lógica do cálculo.
* Manipulação de classes CSS via JS para destacar o resultado.
* Validação básica de campos de entrada.
